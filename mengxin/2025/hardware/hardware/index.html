任选一条产品线中的一台笔记本，讲讲其硬件配置的合理/不合理性，主要集中在 CPU 和 GPU 层面，并适当说明其余硬件。

  1. 请你指认出其中的部件。
  2. 尝试对该电脑的内部设计进行评价（说说他的设计有什么优点（和/或）缺点）。


请简述给笔记本电脑清灰、换硅脂，内存条的操作方法，并指出哪些步骤值得注意。



请简述给电脑加装/更换固态硬盘的方法。（要求从安装到能正常在资源管理器中使用）



你知道引导分区是什么吗？请简述该分区的作用。（需包含不同的操作系统）

当该分区损坏时，我们应该如何修复？



你知道 CPU 物理层面的结构吗？CPU“缩缸”又是什么？CPU 为何会缩缸，我们应该如何避免其发生？如果已经发生，我们又该如何应对？


下图是一张房屋设计图，请回答以下问题：

  1. 卧室的网口出了故障，请问我们应该使用什么工具来进行故障排查？该工具又是如何排查故障的？
  2. 现在该房屋需要做到 WiFi 全覆盖，我们应该如何安排路由器的位置？请以具体的路由器举例。


你的好兄弟小余开学前想整一台平板电脑，于是父母给了他最爱的 HUAWEI MatePad 11 英寸 2023 款柔光版 （祖传 V、落灰 V、死机诅咒），搭载了 HarmonyOS 3.1 系统，却没想到这是恶梦的开篇。

  1. 小余看到满身是灰的平板，决定使用 砂轮 酒精湿巾 好生擦拭，却又担心酒精会不会搞坏屏幕，于是向你求教，你能帮帮他吗？（需要说明具体哪些屏幕可以用酒精擦，哪些不可以）
  2. 平板焕然一新后，小余激动的按下开机键，欸↑，您猜怎着儿？平板没电了……虽然给平板充电并没有技术含量，但好心的你还是决定给 老 baby 好兄弟提点建议。（数量三条及以上，内容以电池寿命为主）
  3. 100%！平板成功开机，眼下只剩最后一道屏障：锁屏密码，由于祖传 V 的特性，小余需要在忘记 锁屏密码、华为账号的密码，且丢失平板包装盒、购买记录的情况下解锁平板。兄弟，救救！


解决了平板，你的好兄弟小余又开始购买电脑。他打开某红色购物软件，搜索”笔记本电脑“五个大字，琳琅满目的商品让他目不暇接。其中，两个便宜且”大牌“的”爆款“映入眼帘：

鉴于之前你对他的帮助，你的兄弟现在非常的膜拜且信任你，于是他选择将这两张图片发给你来做判断。

  1. 这两款电脑，真的是物美价廉的好机器吗？如果不是，请给出合理的解释。

听了你的分析，小余决定先等一等。但是！暑假！就是用来玩的！小余把你偷偷拉到一边，从兜里掏出过去三年辛辛苦苦攒下的 800 块钱，并说：“我的暗恋女神 （妈妈） 叫我一起和她打瓦，但是我没有一台能用的电脑，之前的笔记本也太老了。这些钱给你，帮我配一台台式机，玩过这个暑假就可以。还有鼠标键盘什么的……哦对了，她说打瓦要一百多帧才可以，显示器能不能也算进去……我真的很喜欢她……帮帮我喵……”

你本想拒绝，但看在他是你好兄弟的份上，你接过了这八百块钱。

  1. 你能用这800块钱，配出一台符合情景描述的台式机吗？

  • 要求：主机+显示器+其他外设的总价格不超过800元，能以1080p100帧运行端游”无畏契约“，鼓励尽量省预算、提高性能或者增加硬件稳定性。

  • 请列出详细配置单，包括你所采用的硬件的型号及其市场价格。



转眼，暑假结束了。上次的装机过程中，他学到了许多。你好兄弟追女神 （妈妈） 的后续，他也没有再告诉你。不过，他突然给你发来了微信。原来是因为路途遥远，台式机不方便携带，他斥巨资购入了拯救者的首款商用工作站：拯救者·商：

作为拯救者最豪华的工作站，这电脑可谓是堆料拉满。不过，面对电脑后面的一系列接口，他犯了难：“这……不都一个样吗？还有，这上面的小人，都是什么意思啊……”

解答你好兄弟的困惑罢！（请解释接口上方不同标志的含义）



到了学校不久，哥们打瓦 （找妈妈） 的欲望重燃，在宿舍里开始了打瓦之路。显然哥们已经不能满足于笔记本电脑的小屏幕了，于是自己省吃俭用攒出来些经费给自己安排了一台独立显示器（Type-C 接口），但是自己的笔记本电脑没有全功能的 C 口，所以只能使用 HDMI 的连接线。但这时，哥们突然想起自己似乎是有一个一端是 HDMI，另一端是 Type-C的东西，但是连接上之后并没有成功点亮屏幕。哥们又只能来麻烦你，想问问是怎么个事。

  • 这个东西是什么，为什么不行？
  • 用什么才能正常点亮？
  • 给哥们购买这类转接线的时候提供建议（从数据流角度来看）。


后来，你的名气在哥们之间越传越大，你身边更多的哥们找到了你，你也靠着佣金和辛苦费小赚一笔。有这么一位哥们，在家里经过一番吉列的豆蒸之后和父母谈下了 8500 的预算，想找你提供一整套的方案。（这哥们倒是说自己什么都会一些，淘二手，组装整机都能自己解决）

  • 预算需要包含至少一台高性能 PC，一台移动设备（包括但不限于轻薄本，平板等设备）
  • 最好能实现远程的操作和控制功能，并给出使用方案（以低延迟和 低费用 免费为目标）
  • 支持为了提升性价比的二手以及改装，但不支持以炸死哥们为目的的硬改操作
  • 说出整套方案的优势和面向的未来扩展性，要求 有理有据，文体不限，诗歌除外 能说服你哥们
  • 校园网是局域网 （你知道的太多了）


买电脑只是开始，各种小故障往往更令人闹心。这天，你突然收到哥们的消息，哥们说自己的电脑连不上网了，希望能帮自己看看是怎么个事。于是啪一下，很快呀，就发过来两张 明明是三张 截图：

（感谢@吉吉如律令提供的试题情景）

现在，由于哥们在千里之外，你得通过视频电话的方式远程指导你的哥们。

请简述故障的可能性原因，并且简述维修过程。



😦 你的设备遇到问题，需要重启！

还记得那个不支持被炸死的哥们吗，又是他。一道刺眼的蓝，进入了你哥们的眼睛。伴随一丝不解，你哥们决定先重启电脑。经过了漫长的检查之后，电脑看起来正常了，你哥们也就没在意。但不久之后，蓝屏伴随着开机时的 “no bootable device” 仍然困扰着你哥们：

你哥们有一种不详的预感，于是打开手机浏览器开始 删浏览记录 搜索相关信息，但无论你哥们怎么尝试，蓝屏依旧。他无奈之下向你求助。

  1. 根据题目提供的信息，尝试定位问题所在。
  2. 应该如何解决？


解决故障后，电脑恢复正常。但是，日渐增长的游戏体积和国产 3A 级别大作“小而美”使他的硬盘剩余空间告急。眼看着 C 盘变红的局势已经不可逆转，你哥们决定换一块更大的系统盘。无奈，作为大一学生的他囊中羞涩，特别是刚刚修好电脑，手里更是所剩无几，只能去某黄色软件购买二手盘，可二手盘的价格仍然稍高于他的心理预期，于是他犹豫再三。突然有天晚上，他刷到了一个商品，让他眼前一亮：

150！1TB！你哥们看着这逼真的描述，没多想就下了单。两天过后，硬盘到祸了：

  1. 根据题目所给信息，判断这块盘的真伪（要求说明从哪里得出那些结论，进而证明你的观点）。
  2. 假如这块盘现在就在你手上，你还能做什么来判断这块盘的真伪（要求说明使用什么进行什么操作）。
  3. 在二手平台购买商品时，我们如何买到正品/避免买到假冒产品？收货时应当注意什么？
  4. 如果确实买到了假冒产品，如何进行维权？（尽可能多的列出可能的维权操作）


过了不久，某个哥们的电脑由于操作不当，里面寄生了不少下崽器，终于受不了满屏一刀 999 广告的哥们找到你，想让你帮他清掉这些广告。你看了看，决定还是重装系统，一劳永逸。于是，你准备了一个系统文件，着手开始帮兄弟重装系统。不久之后，你这个系统镜像似乎有点异常，但是没有多想，还是继续安装了。

要求：下载提供的 Windows 11 系统镜像文件，并自行准备一个虚拟机，选用你喜欢的方法，将系统安装到虚拟机中，并对系统进行激活。上面的一切完成后，打开桌面上的交卷程序，完成这道题。

  1. 必须要使用我们提供的镜像文件，此镜像经过特殊处理，同时不建议安装在实体机上
  2. 系统需要完成激活
  3. 每名同学只需提交一次，再次提交记录会被忽略
  4. 若存在非本人提交记录，请联系管理员删除
